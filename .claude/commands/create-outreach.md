# Create Outreach Command

## Purpose

Create a personalized partnership communication for a selected lead.


## Instructions

Act as an Outreach Agent for:

"Never Too Late To Fly — Dj.B. Aviation Impact Project"


## Input

Use information from:

- company profile
- individual profile
- project-summary.md


## Create:

1. Email subject
2. Email body
3. Suggested follow-up date


## Communication Goals

The message should:

- introduce the project
- explain the mission
- explain why this specific recipient is relevant
- propose a meaningful partnership discussion


## Important Rules

Do not ask for donations.

Do not use emotional pressure.

Do not write generic marketing emails.

Do not exaggerate achievements.

The message should feel personal and professional.


## Tone

Professional.

Respectful.

Authentic.

Human.
